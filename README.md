# Image Gallery

A responsive, interactive image gallery web application with filtering and lightbox functionality.

## Features

- **Image Filtering**: Filter images by category (All, Nature, City)
- **Lightbox View**: Click any image to open a fullscreen lightbox viewer
- **Navigation**: Use previous/next buttons to browse images in lightbox mode
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Auto-looping**: Navigation automatically loops back to the first/last image

## Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Modern styling with Grid layout and transitions
- **JavaScript**: Interactive functionality (filtering, lightbox navigation)

## File Structure

```
ImageGallery/
├── index.html      # Main HTML file with gallery markup
```

## Getting Started

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/ImageGallery.git
cd ImageGallery
```

2. Open in browser:
   - Simply open `index.html` in your web browser
   - Or use a local server: `python -m http.server 8000`

## Usage

- **Filter Images**: Click the filter buttons at the top to view images by category
- **View Lightbox**: Click any image to open the lightbox
- **Navigate**: Use the ❮ and ❯ buttons to browse images in lightbox
- **Close**: Click the × button to close the lightbox

## Project Details

The gallery uses placeholder images from Picsum Photos API. Each image is tagged with a category:
- **Nature**: Landscape and outdoor photography
- **City**: Urban and architecture photography
